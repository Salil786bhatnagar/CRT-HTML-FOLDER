Windows-1252

1. Windows-1252 was the first default character set in Microsoft Windows.
2. It was the most popular character set in Windows from 1985 to 1990.

ANSI


1.Historically, the term "ANSI Code Pages" was used in Windows to refer to non-DOS character sets.
2.The intention was that these character sets would be ANSI standards like ISO-8859-1.


Windows-1252 and ASCII


1.The first part of Windows-1252 (entity numbers from 0-127) is the original ASCII character-set. It contains numbers, upper and lowercase English letters, and some special characters.
