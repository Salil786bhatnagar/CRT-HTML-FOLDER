HTML Symbol Entities


1. HTML symbols like mathematical operators, arrows, technical symbols and shapes, are not present on a normal keyboard.
2. To add these symbols to an HTML page, you can use the HTML entity name.
3. If no entity name exists, you can use the entity number.
4. If the character does not have an entity name, you can use a decimal (or hexadecimal) reference.
