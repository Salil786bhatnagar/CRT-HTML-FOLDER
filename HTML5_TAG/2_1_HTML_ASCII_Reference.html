HTML ASCII Reference


1. ASCII stands for the "American Standard Code for Information Interchange".
2. It was designed in the early 60's, as a standard character set for computers and electronic devices.
3. ASCII is a 7-bit character set containing 128 characters.
4. It contains the numbers from 0-9, the upper and lower case English letters from A to Z, and some special characters.
5. The character sets used in modern computers, in HTML, and on the Internet, are all based on ASCII.
6. The following tables list the 128 ASCII characters and their equivalent number.
